# 引き継ぎ

このプログラムは kintone のアプリの設定やカスタマイズファイルを一括ダウンロードするプログラムです。

## 合わせて導入を想定するプログラム

- WinMerge（Windowsソフト）
- Open in External App（Chrome拡張）
- JS Global Viewer（Chrome拡張*別レコード参照）
- cli-kintone
  - グローバルインストールをする
  - アプリ直下のフォルダで vscode を開き ctrl + shift + b で修正ファイルのアップロードが可能になる

## 使用用途の想定

- kintone アプリの最新の設定情報（アクセス情報や通知設定など、APIで取得できる情報）を網羅的に確認する
- WinMerge で、検証環境と本番環境の差異を確認する
  - .winmerge比較用 のフォルダで比較する
